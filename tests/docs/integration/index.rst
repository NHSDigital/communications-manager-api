Integration test suite
======================

The Integration test suite is a collection of tests ran against the integration environment. These tests are ran as part of the `release pipeline <https://dev.azure.com/NHSD-APIM/API%20Platform/_build?definitionId=620>`__ and the `nightly integration integration test run <https://dev.azure.com/NHSD-APIM/API%20Platform/_build?definitionId=628>`__
