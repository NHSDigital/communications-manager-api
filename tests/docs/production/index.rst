Production test suite
=====================

The Production test suite is a collection of tests ran against the production environment. These tests are ran as part of the `release pipeline <https://dev.azure.com/NHSD-APIM/API%20Platform/_build?definitionId=620>`__ and the `nightly production integration test run <https://dev.azure.com/NHSD-APIM/API%20Platform/_build?definitionId=635>`__
